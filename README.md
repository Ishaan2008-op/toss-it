[![HTML](https://img.shields.io/badge/HTML-44.1%25-orange?logo=html5)](https://developer.mozilla.org/docs/Web/HTML)

THIS REPO IS A FULL STACK PROJECT ON THE TOPIC: **IPL MATCH PREDICTION USING THE TOSS INSTANCES AND ENVIRONMENTS**.

## Overview

This project leverages both Frontend and Backend technologies (Python & HTML) to predict IPL match outcomes based on toss events and match environments.

## Features

- Full-stack application
- Data-driven predictions for IPL matches based on coin tosses
- User-friendly web interface

## Getting Started
